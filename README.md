# 🏫 Escuela Deportiva App

Sistema completo de gestión para escuelas deportivas desarrollado con Flutter, Node.js y Turso (LibSQL).

---

# 🚀 Tecnologías utilizadas

- 📱 Flutter (Frontend móvil)
- 🟢 Node.js + Express (Backend API)
- 🗄️ Turso / LibSQL (Base de datos en la nube)
- 🌐 HTTP REST API
